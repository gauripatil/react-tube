# react-tube

ReactTube is a YouTube clone built using React. This application replicates the core features of YouTube, including video streaming, search functionality, user channels, and a responsive user interface designed for an optimal viewing experience across devices.

# Tech stack

- Tailwind - Styling library

# Initial Setup

- Install create-react-app
- Install tailwind `npm install -D tailwindcss@3.2.6`
- Configure tailwindcss - `npx tailwindcss init`
- Update the tailwind.config.js - `content: ["./src/**/*.{js,jsx,ts,tsx}"],`
- Update the App.js -

```
@tailwind base;
@tailwind components;
@tailwind utilities;

```

# Features

    - Hamburger
    - Sidebar (expand/collapse)
    - Search
